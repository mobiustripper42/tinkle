## Task 1: Clock — NTP sync + free-running millis() fallback, local-epoch seam (#26)

**Completed:**
- New platform-independent `Clock` over an injected `IWallClock` seam — `src/core/clock.{h,cpp}` — mirroring the Persistence/ValveDriver core·shim·fake idiom. Anchors a synced reading to a `millis()` instant and free-runs between reads; brisk 1 s retry until lock-on, then a 1 h re-anchor so free-run is exercised and NTP corrects drift.
- **DEC-009 local-epoch seam:** `IWallClock` hands core *local* epoch seconds, so timezone + DST stay in the ESP32 shim and core is a pure, host-testable epoch→fields derivation. `src/esp32/system_clock.h` (`SystemClock`) is the only SNTP/timezone code — `configTzTime` (US-Eastern POSIX TZ + `pool.ntp.org`/`time.nist.gov`), `localtime_r` re-packed via the pure `epochFromCivil` (ESP32 newlib lacks `tm_gmtoff`).
- Exposes `valid()` (→ display `clockValid`), `wall()` (HH:MM + weekday), `epoch()`, and a `minuteRolled()` per-minute eval edge for the Scheduler (#27).
- `main.cpp` — wired to the display, retiring the hardcoded `di.clockValid = false`. Idle shows HH:MM once NTP syncs, `--:--` until then. `systemClock.begin()` + `wallClock.begin()` in `setup()`; `wallClock.tick()` in the loop.
- 7 new native tests + a `FakeWallClock` in `test/test_native/test_main.cpp`: invalid-until-sync, field derivation, free-run between syncs, throttled-resync-then-correct, lock-on-when-source-appears, `epochFromCivil` round-trip (incl. leap day), `minuteRolled` edge.

## Task 2: Scheduler — minute-keyed eval, fert policy, IRunSink seam (#27)

**Completed:**

- **DEC-010 `IRunSink` seam** (`src/core/run_controller.h`): callers that only *request* runs (Scheduler now, web API Phase 4) depend on a narrow `requestRun()` interface; `RunController` implements it. Matches the `IGpio`/`IKeyValueStore`/`IWallClock` idiom — scheduler logic host-tested against a `FakeRunSink` in isolation.

- **Idempotent eval**, forward-only (`curMin <= lastEvalMin_`): each absolute local minute evaluated at most once ever — drops the DEC-009 hourly-resync backward nudge even across a minute boundary.
- `main.cpp` — `scheduler.tick()` wired after the clock (no-op until populated / clock valid).
